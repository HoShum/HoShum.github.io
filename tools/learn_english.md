# 关于本文档
本文档主要用来记录平时看文档遇到不懂的单词，大家平时多多少少都会去看一些官方文档，而并不是所有官方文档都是有中文版的，因此熟悉多点英文单词能有助于我们去理解和学习计算机相关的资料，估制作次文档 \
本文档记录的格式为：单词、翻译、上下文
> 其实并不需要去记忆这些单词，只需要多看看多熟悉即可，尤其要联系上下文，因为很多词汇在计算机的语境中含义是发生改变的

# 关于如何记录单词
本文档是基于一个开源项目[沙拉查词](https://saladict.crimx.com/manual.html)来快速记录单词，大家也可以给浏览器装上这个插件，使用它来记录自己遇到不懂的单词
